# 기하의 시작, 왼쪽 or 오른쪽?
점 A, B, C의 좌표가 주어졌을 때, 반직선 AB의 관점에서 C가 AB의 왼쪽에 있는지, 오른쪽에 있는지를 구분하는 하는 것.
=> 벡터의 외적을 통해 판별.

기하란 점에서 시작하여 선, 면으로 확장되기 때문에, 점과 선간의 관계의 문을 여는 이것이 굉장히 중요하다.

## 두 선분의 교차 문제
