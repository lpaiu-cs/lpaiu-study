# Array

- 인덱스를 활용해 접근한다.
- 요소간에 순서가 있다.

## Static Array

- 크기가 고정되어 있는 배열

## Dynamic Array

- 크기가 가변적인 배열
- 배열의 공간이 부족해질 경우, 크기를 두 배로 늘린다.

>**동적배열에서 시간복잡도**
>
>- 접근: O(1)
>- 말단 삽입(append) 시에는 O(1)
>- 임의 위치 삽입(insert) 시에는 O(n) << 배열을 유지하기 위해 최악의 경우 모두 한 칸씩, n번 밀어야 함.
>- 데이터 삭제도 마찬가지로 말단이면, O(1), 임의 위치면 O(n)
