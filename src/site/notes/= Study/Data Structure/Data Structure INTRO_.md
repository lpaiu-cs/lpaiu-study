# 자료구조

자료(data)를 다룬다는 것은 다음 3가지의 '기본행위'를 수행하는 것이다.

1. **접근**
저장된 요소에 어떻게 접근해야 빠른가?
2. **저장**
그렇게 접근하기 위해 어떻게 저장해야 하는가?
3. **삭제**
또 어떻게 삭제해야 하는가?

따라서 자료구조란, 기본행위를 수행하는 성능 즉, 자료를 다루는 방법의 최적화에 관한 문제이다.

>탐색에 관한 방법
>- Sequenctial Search Algorithm
>- Binary Search Algorithm
