# Priorty Queue

대기열의 데이터들을 우선순위대로 계속 정렬한다.
이는 Heap을 통해 효과적으로 구현 가능하다.

- 일반 Queue와의 차이점: **들어온 순서와 무관**하게 우선순위가 높은 원소부터 pop.
