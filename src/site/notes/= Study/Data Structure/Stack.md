# Stack
Last In First Out 후입선출의 형태.
마치 책을 쌓는 듯한 느낌으로 Stack의 최상단, 즉 가장 나중에 들어온 데이터의 위치를 **top** 이라 한다.

### push
top 위에 데이터를 추가

### pop
top에 위치한 데이터를 제거


>재귀호출과 후위표현식의 계산이 stack의 대표적인 예이다.
