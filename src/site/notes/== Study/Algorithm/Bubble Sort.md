수열의 i번 요소와 i+1번 요소를 비교하여 큰값이 오른쪽이도록 바꾸는 행위를 *for i in range(n)* 으로 수행한다.
만약 정렬이 끝나지 않았다면, 이것을 다시 반복한다.

최악의 경우 O($n^2$) 이지만,
최선의 경우 O(n) 이다.
